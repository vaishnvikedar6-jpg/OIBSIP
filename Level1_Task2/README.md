**Portfolio Website **
Oasis Infobyte Task 2**
**Project Overview**
This project is a professional personal portfolio website
developed as part of my Web Development Internship at Oasis Infobyte.
The goal was to create a responsive, visually appealing, and informative
site that showcases my skills as a Computer Engineer.

**Features**
Hero Section: A modern landing area with a dark purple tech-theme.
About Me: A unique professional bio highlighting my interests in Cloud, AI, and Prompt Engineering.
Certifications Grid: A structured display of my professional credentials from AWS, Microsoft, and more.
Internship Experience: A detailed section highlighting my current role at Oasis Infobyte.
Contact Form: A clean UI for networking and collaborations.
Responsive Design: Fully optimized for mobile, tablet, and desktop views.

**Tech Stack**
HTML5: Semantic structure for better SEO and accessibility.
CSS3: Custom styling using Flexbox, CSS Grid, and CSS Variables.
Theme: Modern Dark Purple and White (Midnight Tech Aesthetic).
